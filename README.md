# Monad NFT

A simple ERC-721 NFT smart contract built for learning and experimentation on Monad Testnet.  
This project focuses on clarity, simplicity, and correct ERC-721 implementation.

---

## Overview

This repository contains an ERC-721 compliant NFT smart contract deployed on **Monad Testnet**.  
The goal of this project is to demonstrate the full lifecycle of an NFT contract: development, deployment, minting, and basic documentation.

---

## Why this project?

This project was created to:

- Practice the ERC-721 standard
- Learn how to deploy NFT contracts on Monad Testnet
- Provide a simple and readable NFT smart contract for beginners
- Showcase a complete and working Solidity project on GitHub

---

## Features

- ERC-721 compliant
- NFT minting
- IPFS metadata support
- Tested using Remix + MetaMask
- Deployed and working on Monad Testnet
